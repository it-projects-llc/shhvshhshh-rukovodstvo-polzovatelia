Для того, чтобы воспользоваться API UPS для Odoo, вам понадобится:

  • ID пользователя UPS.com и пароль
  • Артикул UPS
  • ключ доступа

Ключ доступа - это 16-значный буквенно-цифровой код, позволяющий получить доступ к серверам разработки API разработчика UPS.

Создайте учет UPS

Обратите внимание, что только клиенты, расположенные в США, могут открыть аккаунт UPS онлайн. Если вы находитесь за пределами США, вам придется обратиться в службу поддержки UPS, чтобы открыть учет.

Вы можете ознакомиться с дополнительной информацией об открытии учета UPS на своем веб-сайте на странице [UNKNOWN NODE problematic]Как открыть аккаунт UPS онлайн <https://www.ups.com/content/us/en/resources/sri/openaccountonline.html?srch_pos=2&srch_phr=open+ups+account> [UNKNOWN NODE problematic]_

Если у вас еще нет аккаунта UPS, вы можете создать его вместе с вашим онлайновым профилем, выполнив следующие действия.

1. Access the UPS.com web site at www.ups.com, and click the New User link at the top of the page.

2. Click the Register for MyUPS button, and follow the prompts to complete the registration process.

Если у вас уже есть учетная запись UPS, вы можете добавить его в свой онлайн-профиля следующим образом:

  1. Log in to the UPS.com website) using your UPS.com User ID and Password.
  2. Перейдите на вкладку ** Мой UPS [UNKNOWN NODE problematic].
  3. Нажмите на ссылку ** Резюме учетной записи [UNKNOWN NODE problematic].

4. Click the Add an Existing UPS Account link in the UPS Account Details section of the page.

5. In the Add New Account screen, enter the UPS Account Number, Account Name, and Postal Code fields. The country defaults to United States.q

Получите ключ доступа

После того, как вы добавили свой номер учетной записи UPS в свой профиль, вы можете заказать ключ доступа UPS, воспользовавшись приведенными ниже инструкциями.

  1. Go to the UPS Developer Kit web page
  2. Войдите на UPS.com с вашим ID и паролем
  3. Нажмите на ссылку ** Запрос на ключ доступа [UNKNOWN NODE problematic].
  4. Проверьте свою контактную информацию
  5. Нажмите на кнопку ** Запрос на ключ доступа [UNKNOWN NODE problematic].

** Ключ доступа ** будет предоставлено вам на веб-странице, а электронное письмо с ключом доступа будет отправлено на адрес электронной почты основного контакта.
