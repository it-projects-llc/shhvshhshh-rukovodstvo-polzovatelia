Приложение «Основные средства» (Активы) позволяет вам следить за вашим активами, такими как оборудование, землей и домом. Модуль позволяет автоматически создавать ежемесячные записи амортизации, получать плату за амортизацию, продавать или распоряжаться активами и выполнять отчеты о своих активах компании.

Например, вы можете приобрести автомобиль по 36000 долларов (валовая стоимость), и вы планируете амортизировать актив в течение 36 месяцев (3 года). Ежемесячно (периодичность), Odoo создаст амортизационный запись, автоматически уменьшая стоимость активов на 1000 долларов и проводя 1000 долларов в качестве расходов. Через 3 года эти активы составляют 0 долларов (балансовая стоимость) в вашем балансе.

Различные типы активов сгруппированы в &quot;Типы основных средств&quot;, которые описывают, как отклонить актив. Вот два примера типов основных записей:

  • Строительство 10 лет, годовая линейная амортизация
  • Автомобиль 5 лет, ежемесячная линейная амортизация

Определите типы активов

Тип актива используется для настройки всей информации об основных средствах: основные средства и счета амортизации, метод амортизации и тому подобное. Таким образом, консультанты могут настроить типы основных средств, а пользователи могут дополнительно записывать активы, не требуя предоставления любой сложной бухгалтерской информации. Они просто должны указать тип основного средства на счета поставщика.

Вы должны создавать типы основных средств для каждой группы основных средств, которые вы часто покупаете, например &quot;Автомобили 5 лет&quot;, &quot;Компьютерное оборудование 3 года&quot;. Для всех остальных активов можно создать общие типы основных средств. Назовите их по продолжительности актива, например, &quot;36 месяцев&quot;, &quot;10 лет&quot; …

Чтобы определить типы основных средств, перейдите к разделу Configuration ‣ Asset Types

Создайте основные средства вручную

Чтобы зарегистрировать актив вручную, перейдите к меню Adviser ‣ Assets.

Когда ваш актив будет создан, не забывайте подтверждать его. Вы также можете нажать кнопку Расчет амортизации, чтобы проверить амортизацию, прежде чем подтвердить актив.

Совет

если вы создаете актив вручную, вам все равно нужно создать счет поставщика для этого основного средства. Документ основных средств отражать только записи об амортизации журнала, а не те, что касаются счета поставщика.

Создание основных средств автоматически со счета поставщика

Активы могут быть автоматически созданы по счетам поставщиков. Все, что вам нужно сделать - это установить категорию основных средств в строке счета. Когда пользователь проверит счет, основные средства будут автоматически созданы, используя информацию о счете поставщика.

В зависимости от информации о категории основных средств, актив будет создан по проекту или непосредственно проверено [UNKNOWN NODE problematic]. * Легче подтверждать основные средства непосредственно, чтобы вы не забыли подтвердить это позже. (Проверьте поле * Пропустить состояние черновики * в * Категорию основных средств [UNKNOWN NODE problematic]). Создавайте активы в проекте только тогда, когда вы хотите, чтобы ваш советник руководил всеми основными средствами перед их размещением в ваших счетах.

Совет

если вы установили актив на товар, категория актива будет автоматически заполнена в счете поставщика.

Как амортизировать актив?

Odoo автоматически создаст записи об амортизации в нужную дату для каждого подтвержденного актива (но не черновик). Вы можете управлять амортизационной таблицей: зеленая точка означает, что для этого строки создан журнал.

Но вы также можете опубликовать записи журнала до ожидаемой даты, нажав на зеленую точку, заставив создать связан амортизационный запись.

Примечание

В таблице амортизации нажмите на красную точку, чтобы опубликовать запись журнала. Нажмите кнопку: guilabel: [UNKNOWN NODE title_reference] вверху, чтобы просмотреть записи журнала, которые уже размещены.

Как изменить существующий актив?

  • Нажмите на: guilabel: [UNKNOWN NODE title_reference]
  • Измените количество амортизации

Odoo автоматически перечислит новую таблицу амортизации.

Как зарегистрировать ликвидацию или продажу актива?

Если вы продаете или распоряжаетесь активом, вам нужно полностью прекратить этот актив. Нажмите кнопку: guilabel: [UNKNOWN NODE title_reference]. Это действие будет публиковать полные расходы на эти активы, но не будет зарегистрирована сделка по продаже, которая должна быть зарегистрирована через счет-фактуру клиента.
